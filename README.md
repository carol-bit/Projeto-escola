# Projeto-escola
O projeto escola foi feito durante minha breve aprendizagem sobre Delphi.
Neste projeto em delphi, através de classes, é possível realizar cadastro e consulta de alunos e professores no banco de dados(Firebird).
No cadastro de professor, é possível o usuário adicionar os alunos do mesmo com suas respectivas notas.
Também no cadastro de professor  é possível localizá-lo através do nome ou parte do nome e também pelo nome ou parte do nome de um aluno dele.
Ainda no cadastro de professor, há um botão que calcula a média, a melhor e a pior nota de cada aluno.
Foram implementaras algumas classes:
Uma classe abstrata que declara a assinatura de uma função pública recebendo 4 valores;
• Uma classe herdeira que implementa o método calcular, retornando a média dos 4 valores recebidos.
• Uma classe herdeira que implementa o método calcular, retornado o maior dos 4 valores recebidos.
• Uma classe herdeira que implementa o método calcular, retornado o menor dos 4 valores recebidos.
